## 검증된 것 — LLM 없이 (`--selftest`)

루프 기계장치를 결정적 스텁 에이전트(`ScriptedAgent`)로 검증했다. **실 LLM 없이 검증 가능한
부분을 먼저 닫는다** — 이래야 자격증명이 붙었을 때 실패 원인이 루프인지 모델인지 갈린다.

| # | 프로브 | 관측 | 판정 |
|---|---|---|---|
| ① | 정상 경로(검색 → 지목) | `stop=submitted` · `turns=2` · warnings **4건 전달** | PASS |
| ② | **턴 상한**(50턴 스크립트, cap=3) | `stop=turn_cap` · `turns=3` — 무한 루프 아님 | PASS |
| ③ | 침묵(지목도 기권도 없음) | `submitted=null` · `stop=turn_cap` | PASS |
| ④ | 기권 선언 | `submitted={not_found:true,...}` · `stop=submitted` | PASS |

**③과 ④가 다른 값으로 기록되는 것이 핵심이다.** 침묵("손 놓음")과 기권 선언("모르겠다고 말함")을
같은 값으로 적으면 기권 정밀도/재현율(닫는 기준 5)을 잴 수 없다.

**warnings 4건이 에이전트에게 실제로 전달된다**(①). 구 벤치가 버리던 재질의 신호 —
"본문검색 결과는 관련도순이 아님", "쿼리를 좁혀 다시 물을 것" — 가 이제 관측 가능한 층에 들어왔다.

## 종료 조건 설계 — `submit_answer`

리서치 권고 1의 함정("멀티턴이어도 마지막에 top-N 판정으로 퇴화")을 막기 위해 **에이전트가
조문 하나를 확정 지목해야만** 케이스가 끝난다. 지목은 `submit_answer` 도구 호출이고, 채점기는
그 인자(`law_name`+`article_no`)를 정답 튜플과 문자열 비교한다 — **LLM judge 0개**로 채점이 닫힌다.

## 비용 — 실측 불가, 추정만

**`ANTHROPIC_API_KEY` 미설정 + `ant` CLI 미설치** → 실 LLM 왕복을 돌리지 못했다.
`count_tokens` 도 API 호출이라 토큰 실측도 불가. 아래는 **문자수 실측 + 추정**이며 검증되지 않았다.

문자수 실측 (`--selftest` 트래젝토리에서):

| 항목 | 실측 |
|---|---|
| 도구 스키마(매 턴 재전송) | 1,362 chars |
| 맥락 문단 | 36 chars (셀프테스트용 — AR2 실 문단은 더 길다) |
| `search_law` 응답 본문 | 182 chars |
| ㄴ 그중 warnings | 285 chars (4건) |
| **2턴 케이스 누적 입력** | **≈ 2,047 chars** |

추정 (⚠ **미검증** — 한국어 토큰 밀도를 실측하지 않았다):
- 2턴 케이스 ≈ 2k chars → 대략 1.5~3k input tokens, output 수백 토큰
- 20 케이스 × 3회 반복 × 2~4턴 ≈ **120~240 LLM 호출**
- `claude-opus-4-8`($5/$25 per MTok) 기준 **1회 완주 대략 $1~3** 규모로 보이나,
  토큰 밀도·실 맥락 길이·재질의 횟수에 따라 크게 흔들린다.

**이 추정을 예산 판단의 근거로 쓰면 안 된다.** 자격증명이 붙는 즉시 1케이스를 실제로 돌려
`usage` 를 읽고 이 표를 실측으로 교체한다(러너에 `usage` 누적 배선은 이미 있다).

## 정지 — 자격증명 필요

AR1 plan 의 hard-stop 은 "비용 실측이 예산을 넘으면 정지"였는데, 실제로 걸린 것은 그 앞 단계다:
**실 LLM 을 부를 자격증명이 없어 비용을 재지 못한다.**

- `ANTHROPIC_API_KEY`: unset
- `ant auth status`: `ant` CLI 미설치 (프로필 경로도 없음)
- `@anthropic-ai/sdk`: 미설치 (자격증명이 정해진 뒤 설치한다 — 프로바이더가 바뀌면 SDK도 바뀐다)

step-1 의 남은 것은 **실 LLM 에이전트 구현 + 1케이스 비용 실측** 둘뿐이고, 루프·트래젝토리·
상한·기권 구분은 이미 닫혔다. 자격증명이 정해지면 `Agent` 인터페이스 구현 1개를 붙이는 일이다.
