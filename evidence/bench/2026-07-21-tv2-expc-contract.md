# 국세청 예규 전문 조회 계약 — 실측 결과: **없다**

- 일자: 2026-07-21 · milestone: TV2 step-2
- 이 문서의 목적: plan 이 "전문 조회 계약은 이 step 에서 실측으로 확정한다"로 남긴 미지를 닫는다.
- 판정: **전문 조회 경로가 존재하지 않는다.** 메타데이터 출하로 축소하고 원문 링크를 안내한다.

## 검색은 된다 (1,938건)

```
lawSearch.do?target=ntsCgmExpc&query=가산세&display=3   →  CgmExpc.cgmExpc[]  totalCnt 1938
```

| 필드 | 값 |
|---|---|
| 안건번호 | `기준-2023-법규법인-0191[법규과-992]` |
| 안건명 | ’08년 이전 취득 주식에 대해 ‘출연재산가액 1% 이상 직접 공익목적 사용 의무’ 위반시… |
| 해석기관명 / 해석일자 | 국세청 / 2025.05.15 |
| 법령해석상세링크 | `https://taxlaw.nts.go.kr/qt/USEQTA002P.do?ntstDcmId=200000000000012562` |
| 데이터기준일시 | 2025.06.28 |

**안건번호 체계가 그대로 온다** — 실무자가 인용하는 형식 그 자체다. 기재부(`moefCgmExpc`)도
같은 구조로 66건.

## 전문은 안 된다 (2경로 전부 실패)

| 시도 | 결과 |
|---|---|
| `lawService.do?target=ntsCgmExpc&ID=1643869&type=JSON` | **HTML 반환**(JSON 아님) |
| `lawService.do?target=expc&ID=1643869&type=JSON` | `"일치하는 법령해석례가 없습니다. 법령해석례명을 확인하여 주십시오."` |

두 번째 결과가 중요하다: 법제처의 `expc`(법령해석례)와 국세청 예규는 **ID 공간이 다르다.**
같은 자료원으로 묶어 조회하려는 시도는 실패한다.

> 참고로 `expc` 로 "가산세"를 검색하면 totalCnt **0** 이다. 세법 해석 권한은 기재부·국세청에
> 있고 법제처 해석 대상이 아니기 때문으로 보인다(추정). 즉 **국세청 예규를 안 쓰면 세법 해석
> 자료원이 통째로 없다.**

## 그래서 어떻게 했나 — 빈 응답 대신 명시 거절

전문을 부르면 HTML 이 온다. 그걸 파싱해 빈 결과로 흘리면 소비 LLM 은 **"찾아봤지만 내용이
없다"**로 읽는다. 사실은 "여기서는 못 준다"이고, 원문 링크라는 대체 경로가 있다.

```
get_legal_source(source="ntsExpc", source_id="1643869")
  → DETAIL_NOT_AVAILABLE
    "국세청 법령해석(예규)은(는) 전문 조회를 지원하지 않는다. 이 자료원은 법제처 API 에
     전문 조회가 없다(2026-07-21 실측 — HTML 반환). 검색 결과의 `원문링크`
     (국세법령정보시스템)로 원문을 확인해야 한다."
```

`descriptor.detailUnavailable` 이 설정된 자료원은 upstream 을 **부르지도 않는다**(불필요한
호출과 HTML 파싱을 둘 다 없앤다). 전문이 있는 자료원(심판례·`expc`·`decc`·`nlrc` 등)은
이 필드가 없어야 하며, 과잉 차단을 막는 테스트로 고정했다.

## 부수 확인 — `display=1` 단건객체 결함 재현

```
target=ntsCgmExpc&display=1  →  CgmExpc.cgmExpc 가 배열이 아니라 단건 객체
```

UD3 이 위원회 9종에서 고친 것과 **같은 결함**이다. 기존 `toRowArray` 가 흡수하는 것을
fixture 테스트로 확인했다(정규화가 없으면 조용히 0건이 된다).

이 결함이 프로브 1차에서 ID 추출을 조용히 실패시켰고, 그 때문에 "전문 조회 계약 미확인"이
TV2 로 넘어왔다. 결함 자체는 이미 고쳐져 있었고 **프로브가 그걸 몰랐던 것**이다.

## 남는 것

- **구속력 등급**: 예규는 법원을 구속하지 않는다. step-3 에서 `authority` 등급으로 싣는다.
  전문이 없다는 사실과 별개로, **있는 것을 법조문처럼 보이게 주지 않는 것**이 더 중요하다.
- **신선도**: 데이터기준일시 2025.06.28. 그 이후 예규는 없다.
- 국세법령정보시스템 직접 접근(스크래핑)은 이 horizon 범위 밖이다 — 원문 링크로 사람이 간다.
