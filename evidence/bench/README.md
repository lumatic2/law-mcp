## 기준선(2026-07-20) 해석

**도메인별 recall@3**

| 도메인 | recall@3 | 관찰 |
|---|---|---|
| 부동산임대차 | 80% | 쟁점 용어가 법령명에 직접 들어감(주택임대차보호법·상가건물 임대차보호법) |
| 계약민사 | 60% | "민법"이 워낙 커서 본문검색에도 자주 걸림 |
| 가족 | 40% | 민법 조문이 정답인데 가사소송법·가사소송규칙에 밀리는 패턴 |
| **노동** | **20%** | 근로기준법이 정답인 쿼리에서 병역법·선원법·의료법 등이 상위 |
| **형사** | **20%** | 형법이 정답인데 민법·관세법·전파법이 상위 |

**핵심 관찰 — 법령명에 쟁점어가 있으면 맞고, 없으면 틀린다.** 부동산이 80%인 이유는
"임대차"가 법령명에 있어서고, 형사가 20%인 이유는 "정당방위"·"사기"가 *형법이라는 이름*과
아무 글자도 공유하지 않아서다. 즉 현재 검색은 **쟁점 → 법령 매핑을 전혀 못 하고 법령명
문자열 매칭에 의존**한다. ib3 에서 얹은 법령명 토큰 겹침 신호도 같은 한계 안에 있다.

**판례 hit율 80% vs recall@3 44%** — 판례는 잘 찾는데(사건명에 쟁점어가 들어가므로) 법령을
못 찾는다. 리서치에서 관측한 "hit율 착시"와 같은 구조다.

**리서치 실측치(31%)와의 차이**: 리서치는 13개 임시 쿼리, 기준선은 골든셋 25개 dev 항목이라
구성이 다르다. 같은 자릿수(30~40%대)이고 결론(비세무 도메인에서 절반 이상 실패)은 동일하다.
정본 기준선은 이 44.0% 이며, 이후 개선은 전부 이 수치와 대조한다.
