## Contract

- **비조문 행 제외**: `조문여부 == "전문"`(장·절 제목)은 인덱스에 넣지 않는다.
- **가지번호 지원**: `조문번호=258` + `조문가지번호=2` → `article_no="258의2"`, `display="제258조의2"`.
- **빈 본문 조문 제외**: 텍스트가 없는 조문은 버린다(빈 인덱스 항목 방지).
- **영속 색인 없음**: 프로세스 생존 동안만 유지되는 LRU(기본 20법령). 계획 결정 — 영속 색인은
  설치·갱신 부담을 만들어 "누구나 붙이는 MCP" 목표와 충돌.
- 이 changeset 은 **인덱스 구성만** 한다 — 검색·랭킹 반영은 step-2·step-3.

## LB2 가설에 대한 시사

기준선에서 형사 recall@3 가 20% 였던 이유는 "정당방위"가 *형법*이라는 이름과 한 글자도 공유하지
않기 때문이다. 조문 본문까지 내려가니 **정확히 제21조가 잡힌다** — 법령명 문자열에 의존하지 않는
경로가 실재함을 확인. 다만 이것이 recall@3 상승으로 이어지는지는 step-2 에서 dev 골든셋으로
검증하며, 상승하지 않으면 기각 기록 후 정지한다(계획 hard-stop).
