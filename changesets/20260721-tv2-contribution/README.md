## 왜 골든셋 recall 로 재지 않았나

골든셋은 **법령** 정답을 라벨링한 세트다. 심판례·예규는 정답 라벨이 없다. LB3 에서 자료원
기여도를 골든셋으로 재려다 용도 불일치를 이미 배웠다. 그래서 **도달**(관련 문서가 상위 3에
오는가)과 **표기**(등급이 100% 실리는가)를 잰다. 관련성은 질의마다 미리 정한 키워드로
판정한다 — 느슨하지만 일관되다.

## 정직하게 짚은 한계

**도달 100% 는 관련도 순위가 아니다.** 상위 3 제목이 가나다·숫자순으로 온다:

```
기업업무추진비 손금불산입 → 예규 상위 3
  · 1990.01.01 지급이자 손금불산입 대상주식의 장부가액   ← 쟁점 다름
  · 1주당 최근 3년간의 순손익액의 계산방법               ← 쟁점 다름
  · 1회 접대 지출 금액이 5만 원 이상인 경우 손금불산입    ← 관련 있음
```

즉 이 게이트가 확인한 것은 "관련 문서가 상위 3에 **하나라도** 들어온다"이지 "상위 3이 전부
관련 있다"가 아니다. **프리모템 ②(노이즈만 늘어난다)가 완전히 해소된 것은 아니다** —
판정서에 그대로 남겼다. 다만 도달률 0% → 100% 이고 등급 표기로 오인용은 막았다.

## 값이 실제로 나온 사례

`세금계산서 지연발급 가산세` 예규가 **정확히 1건**인데 그 1건이 정확히 그 쟁점이다.
이 질의는 TV1 기준선에서 **법령 검색이 실패한 건**이다 — 법령으로는 못 닿는데 해석자료로는 닿는다.
