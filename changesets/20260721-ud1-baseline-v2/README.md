# UD1 step-3 — 신 세트 기준선 실측 + 노이즈 정량화

- 일자: 2026-07-21 · milestone: UD1 · 증거: `evidence/bench/2026-07-21-ud1-baseline-v2.md`

## 결과 한 줄

**신 세트 dev blind recall@3 = 48.0%** (구 세트 76.0%). 28%p 차이는 도구가 나빠진 게 아니라
**구 세트가 튜닝에 소진됐다는 증거**다 — 48% 가 처음 보는 질의에서의 실제 실력이다.

## 계획이 틀렸던 지점

계획은 "노이즈를 정량화하고 2σ 를 채택 문턱으로 쓴다"였다. 11회 측정 결과 **문제는 노이즈가
아니라 드리프트**였다: 60% 가 6회 연속 → 48% 가 10회 연속, 한 번 내려간 뒤 안 돌아온다.

뒤집힌 쿼리 5건이 전부 **법령명 검색 0건 → 본문검색 추락** 패턴이었다. 유력한 원인은 상류
쓰로틀링(오늘 수천 회 호출)이며, 사실이면 벤치가 아니라 **실사용자 문제**다(F12).

연속 실행의 σ 는 드리프트를 못 잡는다(배치3 σ=0.0%p → 문턱 0 = 무의미). 그래서 **2σ 문턱을
폐기하고** UD2 의 A/B 규약을 교차 측정 + 쿼리 단위 승패로 바꿨다(F13, 증거 문서에 규약 명시).
