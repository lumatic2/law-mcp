## 측정이 뒤집힌 지점 (이 단계의 핵심)

1차로 dev 골든셋 25쿼리를 돌렸더니 도달 100%·기여 0% 라는 이상한 조합이 나왔다. 결과를 눈으로
읽자 원인이 보였다 — 반환 항목이 **전부 ㄱ으로 시작**했다(감사원·강릉시·강진군·경기도…).
본문검색이 관련도가 아니라 가나다순 앞부분을 주기 때문이고, 따라서 **도달 100% 는 사다리가 만든
허수**였다. 수치만 봤으면 "다 도달하는데 기여만 없다"는 엉뚱한 결론을 냈을 것이다.

그런데 이 측정 자체가 불공정했다. 골든셋은 "쟁점 → **법령**" 쿼리인데 이 도구들은 소비 LLM 이
법원을 지목해 부를 때 쓰인다. 법원별 대표 쿼리로 다시 재고, **이름 매칭 단계 도달만** 인정하자
expc 5/5 · decc 5/5 · ordin 5/5 · detc 3/5 · lstrm 3/5 로 나왔다. 결론:
**법원 도구는 쟁점 탐색용이 아니라 지목 조회용이다.** 이 성질을 도구 description 에 박았다.
