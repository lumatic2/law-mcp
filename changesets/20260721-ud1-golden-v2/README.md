## 왜 새 세트가 필요했나

구 `bench/golden.json` 은 더 이상 판정에 쓸 수 없다:

- **dev 25건은 튜닝에 소진**됐다 — LB2·LB5 가 이 25건을 보고 파라미터를 골랐다(F5).
- **holdout 15건은 은퇴**했다 — LB2 에서 1회, LB5 에서 마지막 1회를 썼다.

즉 이 레포는 지금 **측정 불능 상태**다. `aiSearch` 편입(UD2) 같은 큰 변경을 이 위에서 판정하면
LB1~LB5 의 실수를 그대로 반복한다.
