## 이 step 에서 더 고친 것 — 배포 스모크가 잡은 기존 결함

`민법 시행령`(실재하지 않는 법령)의 조문을 물으면 **난민법 시행령** 조문이 조용히 왔다.
`난민법 시행령` 이 `민법 시행령` 을 문자열로 포함해서다. UD4 회귀가 아니라 원래 있던
결함이고, UD0·LB3 `ordin` 과 같은 부류다 — 틀린 답이 성공처럼 온다.

마지막 폴백은 `부가가치세` → `부가가치세법` 같은 조회를 살리는 장치라 없애지 않고,
**어느 법으로 해석했는지 경고로 남긴다.** 조용한 오답을 시끄러운 오답으로 바꾼 것이다.
