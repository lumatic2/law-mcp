## 정직하게 남긴 것

- 재정렬은 **`limit` 으로 잘린 풀 안에서만** 동작한다 — 이득이 `limit=10`(도구 기본값)에서
  나타나고 `limit=3` 에서는 줄어든다. 판정은 기본값 조건에서 했다.
- `신고·불복·기한` 유형(수정신고·심판청구)은 **여전히 실패**한다. 지방세기본법·관세법이 같은
  제도를 평행으로 갖고 있어 쿼리에 "국세"가 없으면 가를 신호가 아직 없다.
- 범용 세트 판정은 절대 문턱이 아니라 **교차 A/B 손실 0** 으로 했다 — 이 세트의 절대 수치는
  드리프트로 84~88% 를 오간다.
